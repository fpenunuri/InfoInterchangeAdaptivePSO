The folder 'Mech_Fun' contains the following functions:

'CouplerPoint4R': The position vector for the coupler-point curve of the 4R mechanism.
Auxiliary functions:
'is2pi_order'
'isCrankCond'
'isGrashof'
'isordered'
'to_cyclic_order'
Type 'help function_name' to display help for each function.

The folder 'ObjectiveF' contains the objective function for the gait rehabilitation
synthesis problem.

The optimization is done in the file 'SinthEjArtPSO4R.m'
